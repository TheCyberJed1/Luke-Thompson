**Focus**: Developing capability to analyze malware samples and understand attacker techniques.

- Dynamic analysis (behavioral observation, API hooking)

- Reverse engineering basics
- Packing and obfuscation techniques

**Current Study**:
- Platform: Hybrid-Analysis.com (public malware analysis)
- Hands-on: Analysis of 10+ real malware samples
- Tools: IDA Free, OllyDbg, Process Monitor, Wireshark
- Goal: Complete GIAC Certified Malware Analyst (GCIH) preparation

**Relevance to SOC**: Supports malware alert investigation and behavioral detection
