Content Manager

Add and manage labs, scripts, and incident writeups dynamically.

How to use this manager:

  1. Use the tabs below to add new Labs, Scripts, or Incidents
  2. Fill in the form and click "Add Item" to add it to the list
  3. Review your items in the list below each form
  4. When ready, click "Download JSON" to save the updated data file
  5. Upload the downloaded JSON file to the /data/ folder in your repository
  6. Commit and push the changes to see them live on your site

Note: This is a client-side tool. Changes are stored in your browser until you download the JSON files.
